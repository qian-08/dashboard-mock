# Dashboard UI (HTML)

シンプルなダッシュボード風UIをHTML/CSSで作成したプロジェクトです。  
データ可視化・管理画面風レイアウトの練習用として制作しています。

---

## Overview
このプロジェクトはダッシュボード風のレイアウトを持つWeb UIです。  
サイドバー・カードUI・統計表示などの基本構造を含んでいます。

---

## Features
- レスポンシブ対応レイアウト
- サイドバー付き管理画面風UI
- カードベースの情報表示
- シンプルなデザイン構成

---

- AI-assisted development

- Main content area
